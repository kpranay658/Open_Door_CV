## ✨ What is OpenDoorCV?

For years, building a clean, genuinely custom resume meant hitting a paywall. Every tool charged ₹800–₹2000/month just to download your own resume.

**So I built the solution.**

OpenDoorCV combines the internet's most effective, recruiter-tested CV prompts into a single, powerhouse AI tool. No subscriptions. No watermarks. No paywalls. Just you and your best resume.

### 🧠 The real difference — the prompts

Anyone can ask ChatGPT to "write my resume." What makes OpenDoorCV genuinely better is that every generation runs through **battle-tested, recruiter-aligned prompts** that have been researched, refined, and engineered specifically to:

- Pass ATS (Applicant Tracking Systems) filters
- Use the right action verbs for each industry
- Quantify achievements the way hiring managers actually read them
- Match the tone and language of the job description
- Flag and fix red flags like gaps and vague job titles

You get Claude-level AI output with expert-level prompting — **without needing to know how to prompt AI.**

## 🚀 How to use

1. Visit **[OpenDoorCV](https://yourgithubusername.github.io/opendoorcv)**
2. Create a free account
3. Get your free Anthropic API key at [console.anthropic.com](https://console.anthropic.com) — new accounts get **$5 free credits (~250 resumes)**
4. Paste your CV details and the job description
5. Hit generate — your tailored resume is ready in seconds

> **Why do I need an API key?**
> OpenDoorCV uses Claude AI (by Anthropic) under the hood. Your key stays in your browser — it's never stored on any server. This is what keeps the tool free for everyone.

This is the **free, open trial version** of OpenDoorCV.

If this project gains traction and real users find genuine value in it, I plan to launch a hosted version where the API costs are covered — at a **nominal charge far below any competitor.** Think ₹99/month or a ₹499 lifetime deal, not the ₹2000/month that other CV tools charge.
